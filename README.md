Uma área de membros moderna e de alta performance construída com Django, focada em entregar a melhor experiência em usabilidade para os alunos e uma gestão eficiente para os administradores.

🚀 Tecnologias Utilizadas
Backend: Django 5 (Python)

Frontend: HTMX, Alpine.js e TailwindCSS (DaisyUI)

Tarefas Assíncronas: Celery com RabbitMQ

Gerenciador de Pacotes: uv

Integrações: Webhooks da Kiwify e envio automático de emails via SMTP

✨ Funcionalidades Principais
Gestão de Cursos e Aulas: Criação de módulos, upload de capas e controle de progresso completo do aluno.

Navegação Fluida: Transições instantâneas entre aulas usando o HTMX, sem precisar recarregar a página inteira.

Automação de Vendas: Recebedor dedicado para processar vendas da Kiwify e matricular novos alunos automaticamente.

Fila de Emails: Disparo de senhas e avisos de acesso usando o Celery para não travar a experiência do usuário na tela.

Busca Inteligente: Pesquisa de aulas em tempo real com menu suspenso interativo.
